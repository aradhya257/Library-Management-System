# 📚 Library Management System

## 🚀 About Project
This is a Library Management System built using Flask.  
It helps manage student records, book issuing, and reports.

## ✨ Features
- Add Student
- Edit Student
- Book Request
- Issue Books
- Dashboard
- Reports

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Python (Flask)
- MySQL

## 📂 Project Structure
- Templates (HTML files)
- app.py (Main backend)
- library.sql (Database)

## ▶️ How to Run
1. Install requirements
2. Run app.py
3. Open browser
